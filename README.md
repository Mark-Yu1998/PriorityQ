# PriorityQ


The priority queue is implemented using the heap data structure, it stores integer that user pass in, and apply different operation on the stored data. The program supports standard queue operation: Enqueue, dequeue, isEmpty. In addition to flushing all the number that are the multiple of a certain number from the user, or/ and remove all the even/ odd number from the heap. The program only stores value that is greater than a certain range, and non-prime
